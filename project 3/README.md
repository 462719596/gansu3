# 墨陇行 - 甘肃国风旅游官网

> 一笔大漠孤烟，千里陇上江南

墨陇行是专注于甘肃深度文化旅游的官方网站，采用现代Web技术栈构建，展现了中国传统美学与现代设计的完美融合。

## 🎨 设计特色

### 国风美学
- **宣纸纹理背景**：使用 CSS radial-gradient 模拟真实宣纸质感
- **传统色彩系统**：墨黑、金色、朱砂、青灰等传统色彩搭配
- **书法字体**：标题采用站酷庆科黄油体，正文使用思源宋体
- **水墨元素**：图标悬浮墨晕扩散效果，卷轴样式组件

### 现代交互
- **流畅动画**：基于 Framer Motion 的精致动画效果
- **响应式设计**：完美适配桌面端、平板和移动设备
- **无障碍设计**：符合 WCAG 标准的可访问性设计

## 🚀 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 5
- **样式方案**：Tailwind CSS 3 + 自定义CSS
- **动画库**：Framer Motion
- **路由管理**：React Router DOM
- **图标库**：Lucide React

## 📱 功能模块

### 首页 (/)
- **Hero区域**：全屏大漠插画配竖排毛笔字
- **快速导览**：四大景区圆形水墨图标
- **精选路线**：横向卷轴样式的路线卡片
- **响应式交互**：滚动触发动画，CTA按钮交互

### 目的地 (/destinations)
- **瀑布流布局**：Masonry网格展示各大景点
- **悬浮效果**：鼠标悬浮放大并显示详情
- **筛选功能**：按景点类型和地区筛选
- **详情页面**：景点详细介绍和实用信息

### 精选路线 (/itinerary)
- **时间轴设计**：左侧墨线，右侧路线卡片
- **手风琴展开**：点击查看详细行程安排
- **价格对比**：清晰的价格和服务对比
- **在线预订**：直接跳转预订系统

### 画卷长廊 (/gallery)
- **瀑布流相册**：不规则网格展示摄影作品
- **Lightbox效果**：点击图片全屏浏览
- **卷轴边框**：图片外框模拟传统卷轴
- **分类筛选**：按主题和地区分类浏览

### 关于我们 (/about)
- **竖排文字**：传统中文排版方式
- **团队介绍**：核心成员专业背景展示
- **企业文化**：品牌故事和价值观阐述
- **联系方式**：多渠道联系信息

## 🎯 SEO优化

- **页面标题**：每页包含"甘肃旅游"关键词
- **Meta描述**：针对性的页面描述和关键词
- **Open Graph**：社交媒体分享优化
- **结构化数据**：搜索引擎友好的内容结构
- **性能优化**：图片懒加载，代码分割

## 📱 响应式设计

### 断点设置
- **移动端**：< 768px - 纵向卡片布局
- **平板端**：768px - 1024px - 混合布局
- **桌面端**：> 1024px - 横向卷轴布局

### 移动端特性
- **汉堡菜单**：折叠式导航菜单
- **触摸优化**：适合手指操作的按钮尺寸
- **滑动交互**：支持触摸滑动浏览
- **性能优化**：移动端图片尺寸优化

## 🎨 动画系统

### 核心动画
- **页面进入**：fade-in + slide-up 组合动画
- **卷轴展开**：模拟古代卷轴展开效果
- **墨晕扩散**：鼠标悬浮时的水墨扩散动画
- **图片懒加载**：渐进式图片加载动画

### 微交互
- **按钮反馈**：hover、active状态的细微动效
- **页面滚动**：视差滚动和元素进入动画
- **表单交互**：输入框聚焦和验证动画
- **加载状态**：优雅的loading动画效果

## 🚀 部署说明

本项目配置了现代化的构建和部署流程：

### 本地开发
```bash
npm install
npm run dev
```

### 生产构建
```bash
npm run build
npm run preview
```

### Vercel部署
```bash
npm run build
vercel --prod
```

### 部署优化
- **代码分割**：按路由自动分割JavaScript包
- **图片优化**：使用Unsplash CDN，支持动态尺寸
- **缓存策略**：静态资源长期缓存，HTML短期缓存
- **Gzip压缩**：自动启用资源压缩

## 🛠️ 开发规范

### 代码组织
- **组件化**：单一职责的React组件
- **模块化**：合理的文件和目录结构  
- **类型安全**：完整的TypeScript类型定义
- **样式管理**：Tailwind工具类 + 自定义CSS变量

### 性能优化
- **懒加载**：图片和路由的懒加载
- **Tree Shaking**：自动移除未使用的代码
- **Bundle分析**：定期分析打包体积
- **缓存策略**：合理的浏览器缓存设置
